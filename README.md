# The DISMTools Starter Script Library

A collection of starter scripts for anyone to enjoy.

## Contributing to the Starter Script Library

To contribute with your script, use the Starter Script Editor. Once you have your script ready for upload, click "Upload script" and follow the instructions.

## Security implications

Because you're uploading starter scripts for the world to use, try not to upload scripts that contain information used to identify people or company-wide resources. Use the "Inspect Script Security" action in the toolbar to perform an automated inspection for common security violation patterns.
